# @twowayexit/is-even-luau

Roblox TypeScript typings for underonunicom's ~~questionably inefficient~~ [IsEven package](https://github.com/underonunicom/IsEvenLuau)

## Installing

Create a `.npmrc` file in the root directory of your project with this content:

```
@twowayexit:registry=https://npm.pkg.github.com
```

npm: `npm i @twowayexit/is-even-luau`

pnpm: `pnpm add @twowayexit/is-even-luau`

bun: `bun add @twowayexit/is-even-luau`
